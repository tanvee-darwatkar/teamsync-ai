# TeamSync AI 🧠

**Dynamic Team Compatibility & Retention Prediction Engine**

AI-powered platform that helps users build, evaluate, and optimize teams before and during project execution with real-time compatibility analysis.

## 🌟 Key Features

### 1. **Dashboard**
- Team health overview with quick insights
- Team Compatibility Score
- Project Success Probability
- Team Stability Index
- Active team members summary

### 2. **My Profile**
- Complete user profile management
- AI-generated scores (Technical, Communication, Leadership, Collaboration, Reliability)
- Resume/CV upload and analysis
- GitHub profile integration
- Skills and interests tracking

### 3. **My Team**
- Create and manage multiple teams
- Add/Remove team members
- Invite members via email
- **Dynamic recalculation** - AI automatically re-analyzes when members change
- Real-time compatibility updates

### 4. **Team Analysis** ⭐ Core USP
- Compatibility Heatmap
- Skill Coverage visualization
- Leadership Distribution
- Conflict Risk assessment
- Team Strengths & Weaknesses
- Deep intelligence insights

### 5. **Find Teams**
- Discover teams needing your skills
- AI-powered team matching with compatibility scores
- View missing skills and requirements
- Request to join teams
- Personalized match explanations

### 6. **Find Talent**
- Discover candidates for your team
- Skill Match % analysis
- Compatibility % scoring
- Experience analysis
- AI-powered candidate recommendations
- Send invitations directly

### 7. **AI Role Assignment**
- Automatic role assignment recommendations
- Explains why each role was assigned
- Competency scoring for each member
- Optimal team structure suggestions
- Match scores for role fit

### 8. **Skill Gap Detector**
- Identifies missing expertise
- Priority-based gap analysis
- Impact assessment for each gap
- Recommended candidates with match scores
- AI explanations for recommendations

### 9. **Predictions** 🚀 Innovation Showcase
- Project Success Probability
- Team Retention Forecast
- Conflict Risk Prediction
- 6-month trend analysis
- Risk assessment and mitigation strategies
- Future outcome analysis

### 10. **Reports**
- Generate professional reports
- Team Evaluation Report
- Compatibility Report
- Skill Gap Report
- Prediction Report
- PDF download option
- Email delivery and shareable links

## 🎯 Unique Value Propositions

Unlike traditional team-building platforms, TeamSync AI:

✅ **Predicts project success** before team formation
✅ **Predicts team retention** to prevent turnover
✅ **Detects conflict risk** proactively
✅ **Dynamically re-analyzes** teams when members change
✅ **Matches users with teams** bidirectionally
✅ **Matches teams with users** for talent discovery
✅ **Explainable AI** - every recommendation includes reasoning

## 🛠 Tech Stack

### Frontend
- **React 18** - Modern UI framework
- **Vite** - Lightning-fast build tool
- **Tailwind CSS** - Utility-first styling
- **Recharts** - Data visualizations
- **React Router** - Client-side routing
- **Lucide React** - Beautiful icons

### Backend (Ready for Integration)
- **FastAPI** - Modern Python API framework
- **spaCy** - NLP for resume parsing
- **PyPDF2** - Resume/CV processing
- **GitHub API** - Activity analysis
- **Firebase** - Real-time database (planned)

## 📦 Installation

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build
```

## 🚀 Quick Start

1. **Login/Signup** - Create your account
2. **Complete Profile** - Add skills, interests, GitHub profile
3. **Create or Join Teams** - Build your team or discover compatible ones
4. **Get AI Insights** - Receive real-time compatibility analysis
5. **Optimize** - Use predictions and recommendations to improve

## 🌐 Deployment

Deploy to Vercel in seconds:

```bash
npm install -g vercel
vercel login
vercel --prod
```

Or connect your GitHub repo to Vercel for automatic deployments.

See [DEPLOYMENT.md](DEPLOYMENT.md) for detailed instructions.
